# 中文分层练习网站 MVP

这是第一个可运行 MVP：老师创建任务包，学生用班级 code 和姓名完成练习，系统保存结果，老师查看基础数据。

## 本地预览

当前环境没有 `npm`，所以项目做成零依赖静态单页应用。运行：

```bash
node server.mjs
```

然后打开：

```text
http://127.0.0.1:4174/
```

未配置 Supabase 时，网站会使用本地演示数据。学生本地演示入口：

```text
班级 code：G7CN
学生姓名：Kevin 或 王明
```

老师后台本地演示登录：

```text
账号：teacher
密码：teacher
```

## Supabase 接入

1. 在 Supabase SQL Editor 运行 `supabase/schema.sql`。
2. 在 Supabase Auth 里创建老师账号。
3. 登录老师后台，打开 `系统设置`。
4. 填入 Supabase project URL 和 anon/public key。
5. 用老师邮箱和密码登录后台。

匿名学生不会直接读取全部班级、任务或题目表。学生必须先提交有效班级 code，并且姓名必须在老师后台的授权学生名单中，前端才会通过 Supabase RPC 读取该班级、该 Level 下已发布任务。

## 已实现范围

- 学生首页和三个学习路径入口。
- 学生用班级 code + 姓名进入。
- 按班级、学习路径、发布状态显示任务包。
- 登录后先选择 Level，再看到 task pack 列表；每个 pack 显示标题、话题、练习次数、简介、题目数量和最近成绩。
- 支持题型：
  - 拖拽对应题
  - multiple choice 选择题
  - fill in the blank 填空题
  - sentence ordering 词语排序题
  - sentence building 拼装句子
  - open response 开放回答题
- 客观题即时反馈。
- 完成页显示总分、正确率、完成情况。
- 完成后可点击右上角 `再次做题`，再次提交会更新该学生该任务包的最新成绩和尝试次数。
- 老师登录、班级管理、任务包管理、题目管理。
- 发布和取消发布任务包。
- 学生完成记录、最近得分、每道题正确率、常见错误题目。

## 题目编辑规则

- 选择题：选项每行一个；正确答案直接写完整选项。
- 填空题：正确答案直接写文本。
- 对应题：选项每行写 `左边=右边`；正确答案可留空，默认使用选项对应关系。
- 排序题：选项每行一个词；正确答案用 `|` 分隔，例如 `我|今天|很|高兴`。
- 拼句题：选项每行一个词；正确答案用 `|` 分隔。
- 开放回答题：不自动计分，只保存回答。
